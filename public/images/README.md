# 画像ファイル配置ガイド

このフォルダには、以下の画像ファイルを配置してください。

## 必要な画像ファイル

### 1. ロゴ画像
- **ファイル名**: `logo.png`
- **推奨サイズ**: 120×120px（最小）
- **形式**: PNG（透過背景推奨）
- **用途**: ヒーローセクションのロゴ表示

### 2. ヒーロー背景画像
- **ファイル名**: `hero-background.jpg`
- **推奨サイズ**: 1920×1080px以上
- **形式**: JPG または PNG
- **内容**: 六角形グリッドパターン、落ち着いた雰囲気、オレンジのアクセント

### 3. Feature セクション画像

#### 3-1. 地図を「育てる」インターフェース
- **ファイル名**: `feature-map.jpg`
- **推奨サイズ**: 600×400px
- **形式**: JPG または PNG
- **内容**: 暗い地図が徐々に明るくなるイメージ、六角形グリッド

#### 3-2. 「通過」ではなく「滞在」を評価
- **ファイル名**: `feature-stay.jpg`
- **推奨サイズ**: 600×400px
- **形式**: JPG または PNG
- **内容**: 場所で時間を過ごすイメージ、暖かいオレンジの照明

#### 3-3. 生活空間の客観的な把握
- **ファイル名**: `feature-visualization.jpg`
- **推奨サイズ**: 600×400px
- **形式**: JPG または PNG
- **内容**: 生活パターンの可視化、データビジュアライゼーション

## 画像の作成方法

1. **自分で撮影・作成**: アプリのスクリーンショットや関連する画像を使用
2. **デザインツール**: Figma、Canvaなどで作成
3. **AI画像生成**: Midjourney、DALL-E、Stable Diffusionなどを使用
4. **フリー素材**: Unsplash、Pexelsなどから適切な画像を探す

## 注意事項

- 著作権に注意し、使用権限のある画像のみを使用してください
- ファイル名は上記の通り正確に命名してください
- 画像の最適化（圧縮）を行うと、ページの読み込み速度が向上します
- WebP形式への変換も検討してください（より高い圧縮率）
